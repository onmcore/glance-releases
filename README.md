## About Glance

Glance is a revolutionary Git GUI designed from the ground up for performance and responsiveness. Whether you're working with large monorepos, massive codebases, or everyday projects, Glance delivers lag-free interactions with a goal of minimal memory usage (targeting 150-200MB in idle state).

### Key Features

- **Ultra-High Performance**: Instant response times even in repositories with millions of commits and hundreds of thousands of files
- **Memory Efficient**: Optimized for minimal memory usage — targeting 150-200MB idle state, focused on lightweight footprint
- **Large Repository Support**: Optimized for enterprise-scale monorepos and massive codebases
- **Intuitive Interface**: Modern, responsive UI that feels native to Windows
- **Complete Git Operations**: Full support for branching, merging, rebasing, stashing, and more
- **Real-time Visualization**: Interactive branch graphs, commit history, and repository status

### Why Glance?

Traditional Git GUIs often struggle with performance as repository size grows. Glance solves this with:

- **Rust-powered backend** using `gitoxide` for maximum efficiency
- **Tauri framework** for a minimal, secure runtime environment
- **Smart caching** and optimized algorithms for lightning-fast operations
- **Direct repository access** without spawning expensive external processes

### Target Users

- Developers working with large monorepos and enterprise codebases
- Teams that value performance and responsiveness in their development workflow
- Power users seeking a modern alternative to traditional Git GUIs
- Organizations concerned with developer productivity and tool overhead
