# Glance 릴리스

Windows용 초고성능 Git GUI — 번개처럼 빠른 저장소 작업과 최적화된 메모리 효율성을 추구합니다.

[English](README.md) | [한글](README.ko.md)

이 저장소는 Glance의 공식 릴리스와 배포 로그를 관리합니다.

## Glance란?

Glance는 성능과 반응성을 최우선으로 설계한 혁신적인 Git GUI입니다. 대규모 모노레포부터 일상적인 프로젝트까지 모든 상황에서 지연 없는 빠른 응답성을 제공하며, 유휴 상태 150-200MB를 목표로 메모리 효율을 최적화합니다.

### 주요 기능

- **초고성능**: 수백만 개의 커밋과 수십만 개의 파일을 가진 대규모 저장소에서도 즉각적인 반응 속도
- **메모리 효율**: 메모리 사용량 최적화 — 유휴 상태 150-200MB 목표, 리소스 효율성 우선
- **대규모 저장소 지원**: 엔터프라이즈 규모의 모노레포와 대용량 코드베이스에 최적화
- **직관적 인터페이스**: 윈도우 환경에 자연스럽게 어울리는 현대적이고 반응성 있는 UI
- **완전한 Git 작업**: 브랜칭, 머징, 리베이싱, 스태싱 등 모든 Git 작업 지원
- **실시간 시각화**: 인터랙티브한 브랜치 그래프, 커밋 히스토리, 저장소 상태 표시

### Glance를 선택하는 이유

기존 Git GUI는 저장소 크기가 증가할수록 성능이 저하되는 문제가 있습니다. Glance는 다음을 통해 이를 해결합니다:

- **Rust 기반 백엔드**: `gitoxide` 라이브러리를 활용한 최상의 성능 구현
- **Tauri 프레임워크**: 최소한의 런타임으로 보안과 효율성 보장
- **스마트 캐싱**: 최적화된 알고리즘으로 번개같은 작업 속도 달성
- **직접 저장소 접근**: 비용이 큰 외부 프로세스 호출 없이 직접 접근

### 대상 사용자

- 대규모 모노레포와 엔터프라이즈 코드베이스를 다루는 개발자
- 개발 워크플로에서 성능과 반응성을 중시하는 팀
- 기존 Git GUI의 현대적 대안을 찾는 파워 유저
- 개발자 생산성과 도구 오버헤드를 고려하는 조직
